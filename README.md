## Prerequisites

- [Node.js](https://nodejs.org/) v18+
- npm (comes with Node.js)

## Setup

```bash
git clone https://github.com/IanChiu-333/INFO_Capstone.git
cd INFO_Capstone/frontend
npm install
```

## Running the dev server

```bash
cd frontend
npm run dev
```

The app will be available at `http://localhost:5173`.

## Building for production

```bash
cd frontend
npm run build
```
